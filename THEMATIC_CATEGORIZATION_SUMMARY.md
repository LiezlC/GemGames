## 🔗 How It Works

### Symbolic Links = Zero Duplication

Each theme folder contains **symbolic links** to games in the main `extracted/` directory.

**Benefits:**
- ✅ Games can appear in multiple themes
- ✅ No file duplication (saves space)
- ✅ Updates to games automatically reflect in all categories
- ✅ Both organizational systems work independently

### Example: "Cosmic Query: Event Horizon"

This game appears in **3 themes**:

1. `content/AI-Aware/Cosmic_Query_Event_Horizon/` → symlink
2. `content/Cosmo_Spacer/Cosmic_Query_Event_Horizon/` → symlink
3. `content/Paint_Splash/Cosmic_Query_Event_Horizon/` → symlink

All three paths point to: `extracted/sonic/Cosmic_Query_Event_Horizon/`
